## HTTP BODY(RFC7230)  

* 메시지 본문을 통해 표현 데이터를 전달한다.   
* 메시지 본문 == 페이로드(payload)   
* 표현은 요청이나 응답에서 전달할 실제 데이터다.   
* 표현 헤더는 표현 데이터를 해설할 수 있는 정보를 제공한다.  
    * 데이터 유형, 데이터 길이, 압축 정보 등등 
* 참고 : 표현 헤더는 표현 메타 데이터와, 페이로드 메시지를 구분해야하지만,

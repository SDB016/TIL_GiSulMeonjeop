# Stable & UnStable 정렬      
  
정렬을 했을 때 중복된 값들의 순서에 따라 아래와 같이 분류        
  
* 변하지 않으면 안정(Stable) 정렬      
* 변하면 불안정(Unstable) 정렬

## Stable 정렬   
  
* Insertion Sort   
* Merge Sort  
* Bubble Sort  
* Counting Sort
